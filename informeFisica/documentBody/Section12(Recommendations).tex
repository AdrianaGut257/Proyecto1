\section{Recomendaciones}

\begin{itemize}
    \item Se recomienda trabajar siempre en el Sistema Internacional de Unidades (SI) para evitar errores en los cálculos y asegurar la coherencia de los resultados.
    
    \item Es importante definir claramente el sistema de referencia y las condiciones iniciales de cada móvil antes de plantear las ecuaciones.
    
    \item Se sugiere realizar primero un análisis conceptual del problema (tipo de movimiento, variables conocidas) antes de proceder con el desarrollo matemático.
    
    \item El uso de herramientas como GeoGebra es altamente recomendable, ya que permite visualizar el comportamiento de las funciones y verificar los resultados obtenidos analíticamente.
    
    \item Para problemas con múltiples móviles, se recomienda representar las ecuaciones de posición en un mismo sistema de coordenadas para identificar de forma clara los puntos de intersección.
    
    \item Se aconseja complementar el análisis algebraico con interpretación gráfica, ya que esto mejora la comprensión física del problema.
\end{itemize}
